Transcodify

Convert any video to the popular H.264 / .mp4 / AAC format

What's this?

This is a very simple service that allows you to convert any video available on the internet into the popular format. Videos converted to this format will play on most modern devices, including mobile phones, tablets, gaming consoles and, of course, PC's.

Right click -> copy link (address / location)

Technical details

  • Convert any format to H.264 / .mp4 / AAC
  • Build using Node.js / fluent-ffmpeg / Express.js / Mustache
  • HTTP only. HTTPS is not supported by FFMPEG.
  • Length of output files is limited to 5 seconds. (demo)
